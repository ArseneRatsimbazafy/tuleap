SVG illustrations that are used for error states should use css class to adapt colors to current theme.

3 helpers are available:

  • tlp-illustration-primary-box used to both stroke and fill a shape.
  • tlp-illustration-primary-dark-box used to both stroke and fill a shape.
  • tlp-illustration-primary-fill used to fill with the primary color.
  • tlp-illustration-primary-stroke used to stroke with the primary color.
  • tlp-illustration-secondary-light-fill used to fill with the secondary light color.
  • tlp-illustration-secondary-light-stroke used to stroke with the secondary light color.
  • tlp-illustration-secondary-dark-fill used to fill with the secondary dark color.
  • tlp-illustration-secondary-dark-stroke used to stroke with the secondary dark color.
  • tlp-illustration-tertiary-fill used to stroke with a light color (mainly used for background elements).
  • tlp-illustration-tertiary-stroke used to stroke with a light color (mainly used for background elements).
